# Laptop Remote Control

This project is a backend server built using Python and Flask.
It allows remote control and communication between devices.

## Tech Stack
- Python
- Flask

## Folder Structure
backend/
├── app.py
├── requirements.txt
└── .gitignore

## How to Run Backend

1. Create virtual environment
2. Install dependencies  
   pip install -r requirements.txt
3. Run server  
   python app.py
