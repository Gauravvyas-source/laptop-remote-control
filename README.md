# Laptop Remote Control (Backend)

This project is a backend server built using Python and Flask.
It provides system-level information and services that can be accessed remotely.

## Tech Stack
- Python
- Flask
- psutil

## Folder Structure

backend/
├── app.py
├── requirements.txt
├── routes/
├── services/
│   ├── system_service.py
│   └── battery_service.py
├── utils/
│   └── response.py
└── .gitignore

## How to Run Backend

1. Create virtual environment (optional but recommended)
2. Install dependencies  
   ```bash
   pip install -r requirements.txt
